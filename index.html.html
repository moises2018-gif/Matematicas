Antes de empezar — dominar los signos

La Regla del Semáforo

Prohíbe la frase "cambia el signo". Diles: "Las ecuaciones son una balanza. Haces la operación contraria en ambos lados."

Ejercicio diario — 5 minutos al inicio

Pon 10 despejes simples. Deben decirlo en voz alta:

x + 8 = 15 → "El +8 pasa restando" → x = 7 x − 3 = −10 → "El −3 pasa sumando" → x = −7 2x = 8 → "El ×2 pasa dividiendo" → x = 4

Truco clave Cuando dominen sin negativos, añade negativos. No al revés.

La regla de oro para el aula

Prohibido decir "despejar" las primeras clases.
Di: "vamos a dejar SOLA a la letra"

La memoria emocional funciona mejor que la lógica. Ponles nombres a las ecuaciones: "Ecuación Patito", "Ecuación Cocodrilo".

Para quitar el miedo a la X y la Y

El Juego de las Cajas — Ponle precio a las letras

Diles: "La X es el precio de una hamburguesa. La Y es el precio de un refresco."

Ejemplo concreto en pizarra

2x + y = 8

→ "Dos hamburguesas y un refresco cuestan 8€"

x + y = 5

→ "Una hamburguesa y un refresco cuestan 5€"

Pregúntales: "Si una hamburguesa y un refresco valen 5, ¿cuánto vale SOLO la hamburguesa?"

Ellos solos dirán: "Pues 5 menos el refresco"

x = 5 − y

✓ ¡Acabas de despejar sin usar la palabra "despejar"!

El cartel de los 3 mandamientos (imprime y pega en el aula)

1

Despejar es dejar la letra sola. Lo que suma pasa restando; lo que multiplica pasa dividiendo.

2

Sustituir es meter un disfraz. Si x = 3−y, donde ponga x, pongo (3−y).

3

El paréntesis se abre en cruz: 2(3−y) = 6−2y. ¡Cuidado con el signo menos!

Sistema de 2 incógnitas

El Método Robot — 4 pasos siempre iguales

No les des el método completo de golpe. Dales una receta de cocina con pasos fijos.

Aviso importante para el docente

No empieces 3 incógnitas si no dominan 2. Cuando dominen el método robot, diles: "Esto es lo mismo, solo hay un paso extra al principio."

La Ley del Embudo

Sistema de 3 incógnitas

La idea: empezamos con 3 incógnitas → reducimos a 2 → reducimos a 1.

3 incógnitas → despejar x → 2 incógnitas (y,z) → ya saben hacerlo

Error #1 — El más común

Olvidar el paréntesis al sustituir

Si x = 3−y, al sustituir en 2x muchos escriben:

MAL: 2 · 3 − y = 6 − y BIEN: 2(3 − y) = 6 − 2y

Regla Siempre escribe el paréntesis. Si no lo escriben, en rojo en la pizarra.

Error #2 — Signo negativo en el despeje

La x está con signo negativo

Si aparece −x, diles que multipliquen TODA la ecuación por −1 antes de despejar:

−x + 2y = 5  → (×−1) →  x − 2y = −5 x = −5 + 2y

Evita Que despejen directamente −x = ... muchos se confunden con el signo final.

Error #3 — Distribuir mal el signo negativo

El "1 fantasma" que salva vidas

Con −3(2−y) casi todos ponen −6 −3y. Enséñales:

−3(2−y) = −1 · 3 · (2−y) → 3·2=6, 3·(−y)=−3y → −1·(6−3y) = −6 + 3y

Regla del semáforo Cuando el número de fuera es negativo, todos los signos de dentro se invierten.

Error #4 — −y = −2, ¿cuánto vale y?

El truco del espejo

Si les queda −y = −2, se bloquean. Diles:

−y = −2  →  (×−1 en ambos lados)  →  y = 2

O la regla callejera: "Si los dos tienen signo menos, se tachan y se vuelven positivos."

Truco para distribuir — Hablar en voz alta

Con 3(5 − 2y) − 4y = 10, diles que lean en voz alta:

"El 3 por +5 es +15"

"El 3 por −2y es −6y"

"Bajo el −4y"

El signo es parte del número, no una operación separada. Que lo digan siempre con el signo incluido.

Progresión para el aula

Los 3 niveles — Solo avanza si el anterior está dominado

1
Nivel 1 Coeficiente 1 directo

x + y = 5 / x − y = 1 → Despejar x en la 1ª y sustituir

▼ ver pasos
De x + y = 5 → x = 5 − y Sustituir en x − y = 1:  (5−y) − y = 1 5 − 2y = 1 → −2y = −4 → y = 2 x = 5 − 2 = 3     Comprueba: 3+2=5 ✓  3−2=1 ✓
2
Nivel 2 Signo negativo al despejar

2x − y = 4 / 3x + y = 11 → Despejar y en la 1ª (cuidado con el −)

▼ ver pasos

El −y molesta. Multiplica la ecuación por −1 primero:

2x − y = 4  (×−1)→  −2x + y = −4 → y = −4 + 2x Sustituir en 3x + y = 11: 3x + (−4+2x) = 11 5x − 4 = 11 → 5x = 15 → x = 3 y = −4 + 2(3) = 2     Comprueba: 6−2=4 ✓  9+2=11 ✓
3
Nivel 3 Multiplicar para crear un 1

3x + 2y = 12 / −x + 3y = 7 → Multiplicar la 2ª por −1 para despejar x

▼ ver pasos

La 2ª tiene −x. Multiplica por −1 para que quede +x:

−x + 3y = 7  (×−1)→  x − 3y = −7 → x = −7 + 3y Sustituir en 3x + 2y = 12: 3(−7+3y) + 2y = 12 −21 + 9y + 2y = 12 → 11y = 33 → y = 3 x = −7 + 3(3) = 2     Comprueba: 6+6=12 ✓  −2+9=7 ✓

Frase motivacional para tus alumnos

"Ustedes no son malos en mates. El sistema de sustitución solo tiene UNA decisión difícil: elegir qué letra despejar. El resto es aritmética de primaria. Si se equivocan, es porque olvidaron el paréntesis. Hoy eso cambia."
