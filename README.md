# MatemáticasBach — Sitio Web de Bachillerato

## Estructura de carpetas

```
bachillerato/
├── index.html          ← Página principal (abrir aquí)
├── css/
│   └── styles.css      ← Todos los estilos (paleta vibrante)
├── js/
│   └── main.js         ← Lógica interactiva (tabs, steppers, accordions)
└── pages/              ← (reservado para páginas adicionales futuras)
```

## Cómo usar

1. Abrir `index.html` en cualquier navegador
2. No requiere servidor — funciona directamente desde el disco

## Cursos

| Curso | Estado | Contenido |
|-------|--------|-----------|
| 1° Bachillerato | 🚧 Vacío | Placeholder listo para rellenar |
| 2° Bachillerato | ✅ Activo | Sistemas de ecuaciones – Método sustitución |
| 3° Bachillerato | 🚧 Vacío | Placeholder listo para rellenar |

## Para añadir contenido a 1° o 3°

Busca en `index.html` el `<div class="curso-frame" id="curso-1">` o `id="curso-3"`
y reemplaza el bloque `.empty-frame` con el contenido igual que en `curso-2`.

## Paleta de colores

- `--c1: #6C3FFF` — violeta principal
- `--c2: #FF4F7B` — rosa acento
- `--c3: #00C9A7` — verde menta
- `--c4: #FFB800` — amarillo energía
- `--c5: #2D2D6B` — azul noche
